Spark_mini_project

Overview
A small mini project demonstrating basic Apache Spark data processing: loading data, cleaning, transforming, and saving results. Intended as an educational example and starting point for experiments.

Prerequisites
- Java 8+ (required by Spark)
- Python 3.8+ (if using PySpark)
- Apache Spark 3.x
- pip packages: see requirements.txt (if present)

Setup
1. (Optional) Create a virtualenv: python -m venv .venv && .\.venv\Scripts\activate
2. Install Python deps: pip install -r requirements.txt
3. Ensure SPARK_HOME and JAVA_HOME are set if running a local Spark installation
